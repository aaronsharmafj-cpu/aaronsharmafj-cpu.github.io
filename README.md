# Aaron Sharma — Personal Portfolio

Personal portfolio website. Built with plain HTML, CSS, and no dependencies.

## Deploy to GitHub Pages

1. Create a new GitHub repo named `<your-username>.github.io`
2. Push this folder to the `main` branch
3. Go to **Settings → Pages → Source → Deploy from branch → main / root**
4. Your site will be live at `https://<your-username>.github.io`

## Local Preview

Open `index.html` directly in a browser — no build step needed.
